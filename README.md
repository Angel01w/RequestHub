# RequestHub

## Frontend
1. Navigate to the `frontend` folder.
2. Run the command to initialize the Vue + Vite + Tailwind project.  
   ```bash
   npm create vite@latest . --template vue
   ```
3. Install Tailwind CSS as per the documentation.

## Backend
1. Setup the ASP.NET Core project (Create a new Web API project).
2. Implement EF Core for SQL Server.
3. Configure JWT authentication with roles (Solicitante, Gestor, Administrador).
4. Implement CRUD operations for Areas, RequestTypes, and Priorities.
5. Establish core Requests workflow and seed data relevant to the application.
6. Add to the `Program.cs` appropriate configurations (including connection string, JWT settings).
